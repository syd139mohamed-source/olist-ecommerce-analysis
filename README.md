# 🛒 Olist E-Commerce Data Analysis

## 📊 Overview
This project analyzes the Olist Brazilian e-commerce dataset to extract insights about sales, customers, payments, and delivery performance.

## 📂 Dataset
- Orders dataset
- Customers dataset
- Products dataset
- Payments dataset
- Sellers dataset

## 🎯 Objectives
- Analyze revenue trends
- Understand customer behavior
- Study delivery performance
- Identify top product categories and cities

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📈 Key Insights
- Seasonal sales trends
- Most profitable categories
- Delivery delays analysis
- Top customer locations
